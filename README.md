# IDU6 - Flight Terminal

专业的单机离线版飞行指令与数据记录终端。专为 iPad 尺寸（特别是 11 寸）与 Slide Over（悬浮窗）模式深度优化，旨在作为 EFB（电子飞行包）的完美伴侣。

## 🎯 设计哲学 (Design Philosophy)
- **最短路径**：消灭一切多余的页面跳转，所有操作在一个平面内完成。
- **肌肉记忆**：采用类似 FMC 的 3D 机械键盘 UI 与青色航电字体显示，防误触设计。
- **绝对离线**：纯前端架构，不依赖任何外部服务器，万米高空切断网络依然满血运行。

## 🚀 核心功能 (Core Features)
1. **关键参数常驻**：Overview (航班信息) / Takeoff (起飞性能) / Landing (落地性能及 VAPP 自动计算)。
2. **沉浸式指令台**：定制化全键盘，包含专属 `Direct To` 图形键与警示色 `DEL` 键。
3. **极速日志流**：点击 CONFIRM 瞬间带时间戳上屏，最新的指令永远置顶。
4. **一键航后打包**：点击 `Pack & Reset`，自动抓取所有输入数据与时间戳，生成 `IDU6_航班号_年月日.txt` 纯文本文件导出并清空面板。

## 📱 部署与安装指南 (Installation)
由于 iOS 系统沙盒限制，请务必严格按照以下步骤将其部署为桌面原生应用：
1. 在 iPad 上使用 **Safari 浏览器** 打开本页面分配的 GitHub Pages 链接。
2. 点击 Safari 右上角的 **分享图标** (方块+向上箭头)。
3. 选择 **添加到主屏幕 (Add to Home Screen)**。
4. 回到桌面，点击 IDU6 图标以全屏/沉浸模式启动。
